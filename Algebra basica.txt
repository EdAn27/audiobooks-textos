“El sistema AudioBooks Access funciona mediante la integración de varias tecnologías que permiten la interacción por voz y la reproducción de contenido educativo. En primer lugar, se utiliza Firebase Firestore como base de datos, donde se almacenan los libros con información como el nombre, la categoría y el enlace del contenido. Luego, el usuario interactúa con el sistema mediante comandos de voz gracias a la API de reconocimiento de voz, por ejemplo diciendo ‘historia’ o ‘matemática’. Cuando el sistema reconoce el comando, realiza una consulta a Firebase para obtener los libros que pertenecen a esa categoría y posteriormente muestra las opciones disponibles al usuario. Una vez seleccionado el libro, el sistema utiliza el enlace almacenado en Firebase para acceder al contenido, el cual se encuentra alojado en GitHub en formato texto mediante enlaces RAW. Después, el sistema descarga ese contenido, lo procesa y lo divide en partes para facilitar su lectura. Finalmente, mediante la API de síntesis de voz, el sistema convierte el texto en audio, permitiendo que el usuario escuche el contenido del libro, además de reproducirlo de forma continua sin necesidad de intervención. De esta manera, el sistema integra base de datos, almacenamiento externo y tecnologías de voz para ofrecer una plataforma accesible, dinámica e interactiva.”
